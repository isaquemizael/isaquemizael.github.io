## Configurando o ambiente

1. Instalar as dependências (vide sessão dependências);
2. Baixe o projeto:

```bash
git clone https://github.com/isaquemizael/isaquemizael-source.git
```


## Como testar

source testar.sh
```

source publicar.sh
```
